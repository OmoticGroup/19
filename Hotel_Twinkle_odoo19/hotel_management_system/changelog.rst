Changelog
=========

1.2.3 (26th Aug'25)
------------------
- **Product Form View** Added warning for the Is Room Type field.

1.2.2 (31st Jul'25)
------------------
- **Required Document Upload during Room Allotment:** Allows for the seamless upload of necessary documents when a room is allocated.
- **Flexible Booking Rescheduling:** Booking tenure can be dynamically adjusted, with automated price recalculations, even after a room has been allotted.
- **Room Exchange with Service Settlement:** Enables room exchanges after any associated paid services have been successfully completed or settled after checkin.
- **Dynamic Extra Guest Charges:** Implements additional charges for guests exceeding the pre-defined base occupancy of a room.
- **Code Optimization & Maintainability:** Significant code refactoring and an improved file structure have been implemented for enhanced readability and long-term maintainability.

1.2.0 (26th Jun'25)
------------------
- **Room Dashboard** Dedicated dashboard for hotel rooms to view bookings, availability, and direct booking actions.
- **Housekeeping** Added Housekeeping team, Housekeeping items, view enhancements, and option to print To do list.
- **Reschedule stay** Reschedule booking tenure if room is not allotted for selected dates.
- **Allotment Restriction** Option to restrict room allotment until the actual check-in time.
- **Code Optimization** Refactored code and improved the file structure for better maintainability.

1.1.2 (09th Apr'25)
------------------
- **Multi Hotel Management** Added Multi Hotel support for website and Hotel Booking .
- **Enhanced Add Rooms feature** More modularity in adding Guests and booking rooms.  
- **Code Optimization** code refracted and improved file structure.  

1.1.1 (28th Mar'25)
------------------
- **Customer Rating Management** Added a rating management system for the hotel bookings.  
- **Enhanced Email Management** More control over the email management.  
- **Code Optimization** code refracted and improved file structure.  


1.1.0 (20th Feb'25)
------------------
- Introduced **Owner Dashboard** for hotel owners to keep track on revenue, occupancy rates, and upcoming check-ins/check-outs.  
- Added **interactive charts like bar, graph, maps etc. along with interactive cards** for data visualization of bookings and many more, allowing filtering by date range.  
- Implemented **Multi-Hotel Management**, enabling users to manage multiple hotel along with their properties under a single system.  
- Allowed separate configurations for each hotel, including room types, pricing, and facilities.
- Improved booking allocation based on property availability.
- Added new `res.config` settings like Auto Invoice generation, Auto Room Selection (for website based on availability), and many more.


1.0.1 (29th Jan'25)
------------------
- Added House Keeping & New hotel service implementation through wizard.
- Improvements in Hotel Services and Facility flow
- Added Groups and access management.

1.0.0 (10th Jan'24)
------------------
- Initial release with hotel booking feature through backend and frontend.
- Added Hotel management for facilities and Services
